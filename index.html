FlowMaid

위 도형 아이콘을 캔버스로 드래그해 노드를 놓으세요
노드에 마우스를 올리면 나오는 초록 점을 드래그해 연결하세요
스페이스 + 드래그로 화면 이동 · 휠로 확대/축소

FlowMaid 코드 (자동 생성) 


      

        
        
      

      

        사용법

        · 도형 아이콘을 캔버스로 드래그 = 노드 추가

        · 노드 드래그 = 이동 / 더블클릭 = 이름 변경

        · 노드의 초록 점을 다른 노드로 드래그 = 연결

        · 노드 선택 후 색상 클릭 = 색 변경

        · 화살표 더블클릭 = 라벨 입력 / Delete = 삭제

        · Space + 드래그 = 화면 이동 · 휠 = 확대/축소
