FlowMaid 코드 (자동 생성) 


      

        
        
      

      

        사용법

        · 도형 아이콘을 캔버스로 드래그 = 노드 추가

        · 노드 드래그 = 이동 / 더블클릭 = 이름 변경

        · 노드의 초록 점을 다른 노드로 드래그 = 연결

        · 빈 곳 드래그 = 영역 선택 (여러 노드 동시 이동·삭제·색상)

        · 노드 선택 후 채우기/테두리 색·스타일 변경

        · 화살표 선택 후 선/끝 모양 변경 (점선·굵게·원·X·양방향)

        · 선택 후 ⊞ 그룹 = 서브그래프로 묶기 / 그룹 더블클릭 = 이름 변경

        · 우클릭 = 상황별 메뉴 (노드·화살표·그룹·빈 곳)

        · 화살표 더블클릭 = 라벨 / Delete = 삭제

        · Space + 드래그 = 화면 이동 · 휠 = 확대/축소
