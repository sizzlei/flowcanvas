# icon — AWS 아키텍처 아이콘 소스

앱의 아이콘 라이브러리에 **기본 제공**되는 아이콘들의 원본(SVG)입니다.
공식 AWS 아키텍처 아이콘 팩(Asset Package, 2025-07-31)에서 48px 라이트 버전과
그룹/카테고리/리소스 아이콘을 카테고리별로 정리해 넣었습니다.

## 아이콘을 추가/갱신하려면

1. 이 폴더 아래에 `.svg`(또는 `.png`) 파일을 추가하거나 교체합니다. (하위 폴더로 분류해도 됩니다.)
2. 프로젝트 루트에서 실행:

   ```
   node tools/build-aws-icons.mjs
   ```

   → `icon/` 전체를 스캔해서 `aws-icons.js`(앱이 로드하는 번들)를 다시 만듭니다.
3. 브라우저에서 `index.html`을 새로고침하면 반영됩니다.

## 참고

- 파일명이 라이브러리 표시 이름이 됩니다. (예: `Arch_Amazon-EC2_48.svg` → `Amazon EC2`)
- `aws-icons.js`는 자동 생성 파일이므로 직접 수정하지 마세요.
- AWS 아이콘은 아키텍처 다이어그램 용도로는 자유롭게 쓸 수 있으나, 아이콘 자체를
  변형하거나 재배포하는 것은 제한됩니다. 개인/내부 용도로 사용하세요.
